# Mini Shell in C++

This project is a simplified Linux shell developed in C++.

## Features
- Execute external Linux commands
- Built-in `cd` command
- Built-in `pwd` command
- Command history
- Exit command

## Technologies
- C++
- Linux system calls
- Process handling with `fork()` and `execvp()`

## Purpose
I created this project to improve my understanding of C++, Linux environments, command parsing, and process execution.
